**Тему работы:** "Создание первого репозитария на GitHub" (условно)
**Цель:** Знакомство с GitHub через создание репозитария, коммитов
**Алгоритм:** Регистрация на сервисе, создание репозитария, внесение изменения в проекте за счёт добавления файла или редактирования исходного.
**Вывод:** В ходе самостоятельной практики дается задача создания публичного репозитария с последующими изменениями в основной ветке GitHub. 
Сама платформа используется для удобства одиночного или командного проекта.
Есть аналоги Obsidian, IT платформа СФЕРА,GitVerse.
